# DAF1 Security Policy

## Supported Versions
We currently support the latest deployed version of the DAF1 smart contract on the BNB Smart Chain (BEP-20).

## Reporting a Vulnerability
If you discover a security vulnerability in the DAF1 project:
- Please report it privately to .
- Do not disclose the issue publicly until it is resolved.
- Include a detailed description, proof-of-concept (if possible), and steps to reproduce the vulnerability.

We take security very seriously and will respond promptly to all reports. Thank you for helping to keep DAF1 secure.
